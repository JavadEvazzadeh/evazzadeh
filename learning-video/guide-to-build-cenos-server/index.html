آموزش ساده و سریع راه اندازی سرور سنت او اس

3040 روز پیش

سلام. تو این آموزش میخوایم یه سرور سنت او اس رو بدون نصب هیچ کنترل پنلی راه اندازی کنیم و سرویس های مورد نیاز اولیه رو روش بریزیم و در نهایت هم بتونیم چند تا سایت رو روش هاست کنیم. سعی کردم که آموزش ساده ای باشه، هرچند لینوکس ساده نیست!

شاید اولین سوال این باشه که چرا بدون کنترل پنل! خیلی راحت کنترل پنل میریزم و این دردسرهارو هم  ندارم. ولی مساله اینه که میشه گفت بدون استثنا تمام کنترل پنل ها منابع زیادی از سیستم رو به اشغال خودشون درمیارن در نتیجه شما علاوه بر هزینه احتمالی خود کنترل پنل، باید هزینه بیشتری هم بابت افزایش قدرت سخت افزاری اون هزینه کنید. البته اگه قصد فروش هاست دارین یا اصلا حوصله سروکله زدن با لینوکس و دستوراتش رو ندارین، کلا نیازی به دیدن این آموزش نیست! از وقتتون استفاده مفیدتری بکنید.

ولی اگه تازه سرور مجازی گرفتید و میخواین تجربیات جدیدی کسب کنین یا میخواین با کنترل پنل ها خداحافظی کنین و به اعماق لینوکس(!) وارد شین، میتونید از این آموزش پنج قسمتی که در مجموع 40 دقیقه از وققتون رو میگیره، استفاده کنین.

یه نکته هم اینکه از این به بعد اسم آموزش هام رو ساده و سریع میزارم و سعی خودم رو میکنم که مثل اسمشون به سرعت و به سادگی آموزش مربوطه رو منتقل کنم.

قسمت اول: آشنایی و توضیحات اولیه

تو این قسمت تنها یه سری توضیحات اولیه رو به صورت صوتی در حدود 5 دقیقه ارائه خواهم کرد.

 

قسمت دوم: اتصال به سرور و نصب سرویس های اولیه

قسمت دوم مربوط آموزش به اتصال به سرور، نصب ادیتور ننو، نصب آپاچی و پی اچ پی و … در ادامه کانفیگ مربوط به آپاچی برای نمایش صحیح و در نهایت تنظیم برای فعال شدن سرویس ها هنگام روشن شدن سرور. در ادامه لیست دستورات اصلی مربوط به این  بخش رو می بینید.

yum install nano
yum install httpd
yum install php
yum install php php-mysql
service httpd restart
nano /etc/httpd/conf/httpd.conf
service httpd restart
chkconfig httpd on

 

قسمت سوم: نصب مای اس کیو ال و اف تی پی

در این قسمت سرویس مای اس کیو ال رو نصب کرده و بعد از نصب مطمئن مای اس کیو ال و تنظیم پسورد، سراغ اف تی پی رفته و بعد از نصب هم تنظیمات مربوطه به سرویس اف تی پی رو روی سنت او اس انجام خواهیم داد.

ظاهرا تنظیمات اف تی پی باعث بروز مشکل در سرور خواهد شد. از اونجایی که این بخش با مای اس کیو ال بود فیلم رو حذف نمیکنم. ولی برای تنظیم اف تی پی از یه منبع دیگه استفاده کنید 

yum install mysql-server
service mysqld start
/usr/bin/mysql_secure_installation
yum install vsftpd
yum install ftp
nano /etc/vsftpd/vsftpd.conf

 

قسمت چهارم: تنظیم دی ان اس در کلودفلر

در این قسمت نحوه کار کردن با سایت کلود فلر Cloudflare  مانند اضافه کردن دامنه و نحوه کار این سایت آموزش داده خواهد شد.

 

قسمت پنجم: افزودن وب سایت جدید به سرور

در آخرین قسمت این مجموعه، نحوه اضافه کردن سایت یا به اصلاح هاست کردن سایت جدید آموزش داده خواهد شد.

cd ~
mkdir -p /var/www/domain.com/public_html
chmod 755 /var/www
nano /etc/httpd/conf/httpd.conf
apachectl -k stop
service httpd start
/var/www/domain.com/public_html/index.html
